# ar-wall

Hokusaiの浮世絵「神奈川沖浪裏」をWebブラウザで表示するサンプルプロジェクトです。

## デモ
https://code4fukui.github.io/ar-wall/

## 機能
- Webブラウザ上で浮世絵「神奈川沖浪裏」を表示
- マウス操作でカメラを制御

## 必要環境
特に必要ありません。Webブラウザがあれば利用できます。

## 使い方
リポジトリをクローンし、index.htmlを開くだけで使えます。

## データ
浮世絵「神奈川沖浪裏」は、メトロポリタン美術館のコレクションから利用しています。
