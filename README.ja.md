# ar-wall

Webブラウザ上で浮世絵「神奈川沖浪裏」を表示するためのサンプルプロジェクトです。

## デモ
https://code4fukui.github.io/ar-wall/

## 機能
- Webブラウザで浮世絵「神奈川沖浪裏」を3D表示
- マウス/キーボードでカメラ操作が可能

## 必要環境
特に必要ありません。Webブラウザがあれば利用できます。

## 使い方
1. [デモページ](https://code4fukui.github.io/ar-wall/)をWebブラウザで開きます。
2. 浮世絵が仮想の壁面に表示されます。
3. マウス/キーボードを使ってカメラを操作し、3Dシーンと対話できます。

## データ
浮世絵「神奈川沖浪裏」は、メトロポリタン美術館のパブリックドメイン コレクションから利用しています。
