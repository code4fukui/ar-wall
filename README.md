# ar-wall

A project to display an image of a famous Japanese artwork as an AR wall.

## Demo
https://code4fukui.github.io/ar-wall/

## Features
- Displays the "Under the Wave off Kanagawa" artwork by Katsushika Hokusai
- Uses Three.js for 3D rendering
- Supports PC control for interaction

## Requirements
- Web browser with WebGL support

## Usage
1. Open the [demo page](https://code4fukui.github.io/ar-wall/) in a web browser.
2. The artwork will be displayed as a virtual wall.
3. Use the mouse/keyboard to control the camera and interact with the scene.

## Data / API
The artwork image is from the public domain collection of the Metropolitan Museum of Art.
